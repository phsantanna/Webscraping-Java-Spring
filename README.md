# Web Scraping com Selenium, Java e Spring Boot

### O projeto surgiu quando eu precisava de novos periféricos para meu computador, porém, precisava que eles estivessem em promoção.
### Então, tive a ideia de fazer um Web Scraping utilizando Java para criar a lógica de negócio,  Selenium e WebDriver para a automação do processo de busca e coleta de dados da plataforma de vendas e o Spring Boot cuidar de armazenar todos os produtos em promoção num banco de dados com o nome, preço e link de cada um para que eu não precisasse olhar produto por produto no site.

#### ** O selenium auxilia no web scraping permitindo fazer uma busca automatizada através de um WebDriver **

#### ** O WebDriver escolhido para auxiliar o processo foi o Edge WebDriver **
